# Good Story Examples

This folder shows how `good-story` behaves on concrete research-writing tasks.

If you are new, start here:

1. Read `before-after.md` to see how weak research text becomes a stronger paper story.
2. Read `field-mini-cases.md` to see how the same story logic changes across ecology, remote sensing, AI4Science, social science, and biomedical research.
3. Copy the prompt shape that matches your task, replace the toy material with your own, and run `$good-story`.

## What These Examples Are

These are compact teaching examples, not claims about real unpublished studies. They are designed to show the move:

`loose material -> story diagnosis -> calibrated story -> rewrite target`

The important part is not the exact wording. The important part is how the skill keeps the story strong without letting it outrun the evidence.

## Example Types

| File | Use it when |
|---|---|
| `before-after.md` | You want to see concrete rewrites of abstracts, result order, and scattered findings |
| `field-mini-cases.md` | You work in a specific field and want to know what must be calibrated first |
