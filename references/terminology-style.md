# Terminology Style

Use this reference when answering in Chinese. The goal is accurate scientific Chinese, not literal word-by-word translation. Prefer a fluent Chinese explanation, with English only when it helps precision.

## Chinese Output Rule

When the user writes in Chinese:

- Use Chinese headings.
- Translate framework terms in the main text.
- Add English in parentheses only on first mention when the term is field-standard or useful for later lookup.
- Keep paper titles, journal names, author names, Latin species names, gene/protein names, and established acronyms in English when appropriate.
- Avoid unnecessary Chinglish in labels and tables.

## Framework Names

Keep framework acronyms when useful, but translate their roles:

- OCAR: 开场、挑战、行动、解决。
- ABT: 已知背景、转折矛盾、因此主张。
- CARS: 建立研究领域、指出研究缺口、占据该缺口。
- Toulmin: 主张、证据、论证桥梁、限定条件、反驳点。
- C-C-C: 背景、内容、结论.

## Before Sending A Chinese Answer

Check for avoidable English labels. Replace them:

- `Best story` -> `最强故事`
- `Why this story works` -> `这个故事为什么成立`
- `Story spine` -> `故事主线`
- `Evidence map` -> `证据地图`
- `Weak points` -> `薄弱点`
- `caveat` -> `限制`
- `claim` -> `主张`
- `turn` -> `关键转折`
- `reviewer` -> `审稿人`
- `overclaim` -> `过度主张`
