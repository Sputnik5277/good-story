# Prompt Templates

Copy one template, replace the bracketed parts, and run `$good-story`.

| Template | Use when |
|---|---|
| `story-diagnosis.md` | You have messy results or a draft and want the strongest story |
| `abstract-rewrite.md` | You want an abstract diagnosed and rewritten |
| `figure-order.md` | You want figures arranged as an evidence ladder |
| `field-calibration.md` | You work across fields or outside familiar examples |
| `rebuttal-prep.md` | You want likely reviewer attacks and repair paths |

For expected output shape, see [docs/OUTPUT_SPEC.md](../docs/OUTPUT_SPEC.md).
